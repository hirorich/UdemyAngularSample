# UdemyAngularSample
[【2021年最新版】はじめてのAngular入門 実践シングルページアプリケーション（SPA）開発](https://www.udemy.com/course/angular-ja/) で配布されたデモアプリ。

## nvm
```
nvm use 18.12.1
```
- 動画内のバージョンより新しくした
- それにともない `package.json` 内の各バージョンも新しくしている

## AdminApp
```
cd admin-app
npm install
npm start
```
- Angularの基礎がわかるサンプルアプリ

## RxjsVisualizer
```
cd rxjs-visualizer
npm install
npm start
```
- Rxjsの動きが視覚的にわかるアプリ
